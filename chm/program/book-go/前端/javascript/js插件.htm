懒加载

  • 插件两个：
    下载jquery-1.7.2.js :1.4版本没用 向上版本没试
    下载jquery.scrollLoading.js:

  • 页面使用：
    <img class="scrollLoading" src="<%=path%>/images/default.png"
    data-url    ="<%=path%>/images/a1.png" width="500" height="180"
    style="background: no-repeat center;" />

    <script>
    $(".scrollLoading").scrollLoading();
    </    script>
    说明 ：所有class为scrollLoading的进入异步加载
    src:默认的一个1*1的图片，进行预加载
    data-url：为真正的图片路径
    height和width为必须的
