----------dwr.xml----------
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE dwr PUBLIC "-//GetAhead Limited//DTD Direct Web Remoting 3.0//EN" "http://getahead.org/dwr//dwr30.dtd">
<dwr>
<allow>
<create creator="new" javascript="DWRUserCheck">
<param name="class" value="com.ysb.UserValidate" />
<include method="checkuser"/>
<include method="checkuserAtter"/>
</create>
</allow>
</dwr>
1.class: 类创建方式
2.javascript: 前台页面的包名；DWRUserCheck
3.param name: 类
4.value:完整类路径
包含; include method
