hadoop 指令 以hadoop2.x为基准

  • 基础指令
    初始化集群 hdfs namenode -format
    集群信息 hdfs dfsadmin -report
    查看文件块信息 hdfs fsck $PATH -files -blocks
    hdfs dfs hdfs dfs -ls [-R]/-cat/-cp/-get/-put/-mv/-rm [-r]/-rmdir/-mkdir/-touchz/-stat/-tail
    权限 hdfs dfs chgro/chmod/chown [-R] <path>
    文件大小 -s 相当于linux dus ; -h 大小的数量级单位 hdfs dfs -du [-s] [-h] $path
    清空回收站 hdfs dfs -expunge
    更改副本系数 hadoop dfs -setrep -w 2 -R /
    文件是否存在 hdfs dfs -test src
    启动job历史记录服务 mr-jobhistory-daemon.sh start historyserver
